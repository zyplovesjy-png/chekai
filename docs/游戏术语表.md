# 扯旋 游戏术语表（已对齐）

> 对齐日期：2026-07-08
> 用途：所有规则讨论、代码命名、注释、UI 文案以本表为准
> 弃用的说法：~~庄注~~、~~开招~~、~~挡家~~、~~圈芒~~（均不使用）

---

## 一、人员与座位

| 术语 | 含义 |
|---|---|
| 庄家 | 每局固定一名，开局交底注 10 分；每局结束后逆时针轮转（跳过破产者） |
| 下家 | 某玩家逆时针方向的下一位（甲是庄家，乙就是甲的下家）；第一轮由庄家的下家先发话 |
| 上家 | 某玩家逆时针方向的上一位（丁是甲的上家） |
| 存活玩家 | 本局还没弃牌、没被淘汰的玩家（含已敲的） |
| 破产 / 淘汰 | 簸簸归零，不能再参加后续局 |

## 二、筹码与金钱

| 术语 | 含义 |
|---|---|
| 簸簸（数） | 玩家当前手里的全部筹码。同时也是单局下注上限——下注不可能超过簸簸 |
| 起簸 | 加入游戏时选择带入的初始筹码。带入 100 即"起簸 100" |
| 加簸 | 游戏过程中追加筹码。**目前代码未实现，待开发**。规则：任意时候（包括输完后）都可发起加簸，但追加的筹码在**当前局结束、结算完成后**才加到簸簸上、下一局生效。例：第三局中簸簸剩 50 时加簸 100，第三局结算后剩 30，则第四局开始时簸簸为 130。目的：防止玩家看牌好临时加簸赢更多 |
| 底注 | 庄家每局开局交的固定 10 分，进底池。不叫"庄注" |
| 皮 / 底池 | 桌面公共池，两个词同义（对应"尾大吃皮"）。构成 = 底注 + 芒果 + 弃牌者支付的钱（+ 上局作废遗留）。喊价中的金额不在底池里 |
| 喊价 / 下注额 | 同义。玩家本局承诺投入的**累计总额**，喊价期间筹码不离手；弃牌时支付自己当前的喊价，比牌时支付最终喊价 |
| 最终比价 | 结算用语：比牌时刻每位存活玩家的最终下注额（敲的玩家 = 敲出的簸簸数）。输家被瓜分的、赢家封顶的都是这个数。底注和芒果不算最终比价 |
| 芒果 | 惩罚性底注。触发后下一局每位玩家（揍芒赢家除外）都要交，进底池 |
| 打芒 | 交芒果的操作。金额 10 = 一芒，20 = 二芒，30 = 三芒 |

## 三、下注动作（发话）

| 术语 | 含义 |
|---|---|
| 发话 | 轮到某玩家表态（下注环节的统称） |
| 过招 | 每轮中所有玩家依次发话下注的整个过程 |
| 休 | 放弃本轮先说话的权利，让后面的人先表态；轮回来后可正常跟/大/敲/丢。仅在本轮还没人抬价时可选 |
| 下注 | 喊价的操作（把自己的喊价抬上去）。没有"开招"的说法 |
| 本轮最低喊价 | 第 1 轮为 10；第 2、3 轮 = 上一轮最终下注额 × 2 |
| 跟 / 瞧 / 买 | 同义。把自己的喊价提到与当前最高喊价一致（本轮下注和上家一致） |
| 大 / 加注 / 返 | 同义。把喊价抬到比当前最高更高的数额，上限为自己的簸簸数 |
| 敲 | 把喊价直接变为自己的全部簸簸数（all-in）；若高于当前最高喊价，视同加注 |
| 丢 / 甩 / 弃牌 | 同义。退出本局，立即支付自己当前的喊价进底池，手牌不亮 |
| 当前注 / 最高喊价 | 场上当前需要跟到的总额 |
| 最终下注额 | 一轮结束时统一的喊价总额，是下一轮最低喊价（×2）的基数 |

## 四、流程与阶段

| 术语 | 含义 |
|---|---|
| 一局 | 从发 2 张暗牌到比牌结算（或作废）的完整过程 |
| 一轮 | 一局中的一个下注环节，最多 3 轮（发 2 张暗牌后、发第 3 张后、发第 4 张后） |
| 暗牌 / 明牌 | 前 2 张只有自己可见；第 3、4 张所有人可见 |
| 配牌 / 分牌 | 比牌前把 4 张牌分成头尾两组 |
| 比牌 | 全员配牌后两两比较定输赢 |
| 全员休 / 休过河 | 任一轮所有存活玩家都休 → 本局作废 |
| 作废 | 本局无输赢：存活玩家的喊价作废、分文不付；弃牌者已付的钱退回；只保留底注+芒果进入下一局 |

## 五、牌型与比牌

| 术语 | 含义 |
|---|---|
| 头牌 | 4 张分成两组后较大的那组（"头大尾小"命名） |
| 尾牌 | 较小的那组；多人比牌顺序和吃底池都看尾牌 |
| 点 | 散牌牌型：两张点数之和取个位；散牌最大 8 点（9 点组合均为独立牌型） |
| 皱 | 0 点，最小牌型 |
| 丁二皇 | 红3 + 大王，最大牌型 |
| 对子 / 杠 / 天九王 / 官九等 | 牌型表中的特殊组合（代码 `COMBO_RAW` 那套） |
| 尾大吃皮 | 尾牌最大者赢得底池（皮），不受封顶限制 |
| 封顶 | 赢家从单个输家身上最多赢"自己的最终比价"，且按输家分别封顶 |

## 六、特殊机制

| 术语 | 含义 |
|---|---|
| 三花十 / 三花六 | 特定三张牌 + 任一张，可亮牌退出本局，免输不赢，退还已投入；亮三花不算弃牌。整副牌最多两人同时成三花（一人十、一人六）。若场上仅剩两人且均摊牌 → 无人收池，底池留到下一局，且不打芒 |
| 休芒 | 全员休导致的芒果惩罚（下一局全员打芒） |
| 揍芒 | 只剩一人赢（其余弃牌结束）：默认下一局除赢家外打芒。赢家未操作别人就丢光，与赢家已下注后别人丢光，效果相同。例外：有人已下注后又面对加价（返）再弃 → 不揍芒。亮三花不算弃牌 |
| 芒果等级 | 连续触发时累加（上限三芒），比牌正常结束后清零 |
| 圈芒 | **不存在此机制，无需实现** |
