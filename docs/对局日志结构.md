# 对局复盘日志（给人看）

> 每场房间对局会生成一份**中文 Markdown 复盘报告**，按局分段写清：谁坐下、谁当庄、发了什么牌、每步操作、比牌/作废结果、整场输赢。
>
> 同目录另有 `debug-*.ndjson`，给程序/排查用，一般不用打开。

## 文件位置

```
logs/rooms/{房间号}/
  复盘-{开始时间}-{短ID}.md      ← 直接打开看这个
  debug-{开始时间}-{短ID}.ndjson ← 机器可读副本
```

已加入 `.gitignore`，不会进仓库。

## 复盘报告长什么样

```markdown
# 扯旋对局复盘 · 房间 452

| 项目 | 内容 |
| --- | --- |
| 总局数上限 | 8 |
| ... | ... |

## 开局名单
| 座位 | 玩家 | 带入 |
| ... |

## 第 1 局
- 庄家：玩家1（user1）
- 底池：10
- 发言顺序：玩家2 → 玩家3 → 玩家1

### 发牌 · 暗牌（前两张）
- 玩家2：黑8、板凳
...

1. **玩家2** 弃牌（未喊价，不扣分） · 手中 500 · 当前最高 0 · 底池 10
2. **玩家3** 叫分 10 · 手中 200→190 · 喊价 0→10 · ...

> 本轮结束：只剩一人，本局结束

### 本局结果：只剩一人，本局结束
输赢：玩家3 +10 · 玩家1 -10

---

## 整场结算
| 玩家 | 总带入 | 最终 | 输赢 |
```

## 术语对照（报告里用的说法）

| 报告用语 | 含义 |
| --- | --- |
| 休 / 弃牌 / 跟 / 叫分 / 加注 / 敲 | 玩家操作 |
| 喊价 | 本局已预扣、尚未进底池的 committed |
| 底池 | potPi（底注+芒果+已弃付等） |
| 全员休，本局作废 | rest_cross，喊价退回，只留底注+芒果 |
| 休芒 N 级 | 下一局全员罚芒 |
| 筹码校验：通过 | 在座筹码+底池 = 总带入（守恒） |

## 旧 NDJSON 怎么转

若手里只有以前的 `session-*.ndjson`：

```bash
node -e "
const { convertNdjsonToMarkdown } = require('./server/game-logger');
console.log(convertNdjsonToMarkdown('logs/rooms/452/session-xxx.ndjson'));
"
```

会在同目录生成 `复盘-*.md`（或覆盖指定输出路径）。

## 隐私

报告含**完整手牌中文名**（含暗牌），仅供本机复盘，勿外传。
